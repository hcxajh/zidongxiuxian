# 修仙游戏核心框架 - 游戏设计文档

## 1. 游戏概述

**游戏名称**: 仙途 (Xian Tu)
**类型**: 文字冒险/放置修仙
**平台**: PC/Mobile
**目标用户**: 修仙小说爱好者、休闲玩家

---

## 2. 核心机制

### 2.1 境界体系
| 境界 | 层级 | 突破要求 |
|------|------|----------|
| 炼气期 | 1-10层 | 灵气积累 |
| 筑基期 | 1-10层 | 筑基丹/顿悟 |
| 金丹期 | 1-10层 | 结金丹 |
| 元婴期 | 1-10层 | 化婴 |
| 化神期 | 1-10层 | ... |

### 2.2 灵根系统
- **五行灵根**: 金、木、水、火、土
- **变异灵根**: 雷、冰、风、暗、光
- 灵根决定功法加成和属性成长

#### 灵根品质
| 品质 | 等级 | 概率 | 修炼加成 |
|------|------|------|----------|
| 天灵根 | 10 | 1% | 1.5x |
| 真灵根 | 9 | 4% | 1.3x |
| 异灵根 | 7 | 20% | 1.15x |
| 杂灵根 | 5 | 25% | 1.0x |
| 伪灵根 | 1-4 | 50% | 0.8x |

#### 灵根效果
- 攻击/防御倍率加成
- 修炼速度加成
- 暴击率加成
- 多灵根协同加成

### 2.3 功法体系
- **基础功法**: 灵气修炼
- **攻击功法**: 术法、神通
- **防御功法**: 护体、结界
- **辅助功法**: 炼丹、炼器、阵法
- **被动功法**: 体质加成

#### 功法品阶
| 品阶 | 星级 | 学习费用 |
|------|------|----------|
| 凡品 | ★ | 50 |
| 灵品 | ★★ | 100 |
| 神品 | ★★★ | 150 |
| 天品 | ★★★★ | 200 |

#### 功法效果
- 等级系统（1-10级）
- 熟练度经验
- 元素契合加成
- 冷却机制

### 2.4 资源系统
- **灵气**: 修炼核心资源
- **灵石**: 货币，可购买药材/功法
- **药材**: 炼丹材料
- **装备**: 、法器、灵宝

## 6. 弟子系统

### 6.1 弟子概述
弟子是玩家在修仙途中收取的门徒，可以帮助玩家进行战斗和修炼。

### 6.2 弟子属性
- **境界**: 炼气期、筑基期、金丹期、元婴期、化神期
- **层数**: 每境界1-10层
- **种族**: 人族、妖族、灵族
- **天赋**: 悟性极高、战斗天才、铜皮铁骨、身法灵动、福缘深厚、寿元悠长
- **忠诚度**: 0-100，影响弟子是否背叛
- **心情**: 0-100，影响修炼效率

### 6.3 弟子功能
- **指导修炼**: 消耗时间指导弟子修炼，获得修为
- **派遣战斗**: 派遣弟子独自战斗，获得战功
- **驱逐**: 可以驱逐不忠诚的弟子

### 6.4 弟子管理
- 玩家最多可收10名弟子
- 招收弟子消耗灵石 (50 * 弟子数)
- 弟子战功可用于宗门建设

## 7. 灵兽伙伴系统

### 7.1 灵兽概述
灵兽是玩家的忠实伙伴，与玩家并肩作战，共同成长。

### 7.2 灵兽类型
| 类型 | 说明 |
|------|------|
| 狼族 | 攻击型 |
| 狐族 | 辅助型 |
| 虎族 | 防御型 |
| 龙族 | 全能型 |
| 凤凰族 | 治疗型 |
| 龟族 | 防御型 |
| 禽族 | 速度型 |
| 蛇族 | 爆发型 |
| 精灵族 | 辅助型 |
| 虫族 | 特殊型 |

### 7.3 灵兽品质
| 品质 | 概率 | 属性加成 |
|------|------|----------|
| 普通 | 40% | 1.0x |
| 稀有 | 25% | 1.3x |
| 精英 | 15% | 1.6x |
| 传说 | 4% | 2.0x |
| 神话 | 1% | 2.5x |

### 7.4 灵兽功能
- **单独战斗**: 派遣灵兽战斗，获得经验
- **联手战斗**: 玩家与灵兽共同出战，伤害叠加
- **训练**: 通过训练提升灵兽经验
- **喂食**: 提升亲密度和饱食度
- **进化**: 积累经验后可进化，属性大幅提升
- **放生**: 可放生灵兽

### 7.5 灵兽技能
- 撕咬、爪击、冲锋、护主、治愈、暴怒、坚固、闪电、冰霜、火焰

### 7.6 灵兽进化
- 每只灵兽有5个进化阶段
- 进化后属性大幅提升
- 进化需要大量经验积累

## 8. 仙侣道侣系统

### 8.1 仙侣概述
仙侣是修仙途中与玩家结为道侣的伴侣，与玩家双修共同提升修为。

### 8.2 仙侣类型
| 类型 | 说明 |
|------|------|
| 修道者 | 同道中人，最常见 |
| 仙人 | 上界仙人，资质极高 |
| 妖魔 | 妖修/魔修 |
| 精灵 | 天地精灵 |
| 鬼修 | 鬼道修士 |
| 龙族 | 龙族伴侣 |
| 凤凰 | 凤凰族 |
| 神秘存在 | 未知来历 |

### 8.3 仙侣资质
| 资质 | 境界 | 属性加成 |
|------|------|----------|
| 凡人人 | 凡人 | 1.0x |
| 炼气 | 炼气期 | 1.2x |
| 筑基 | 筑基期 | 1.4x |
| 金丹 | 金丹期 | 1.6x |
| 元婴 | 元婴期 | 1.8x |
| 化神 | 化神期 | 2.0x |
| 合体 | 合体期 | 2.2x |
| 大乘 | 大乘期 | 2.5x |
| 仙人 | 仙人 | 3.0x |

### 8.4 好感等级
| 等级 | 好感度 | 效果 |
|------|--------|------|
| 陌生人 | 0-20 | 基础双修 |
| 相识 | 21-40 | 修炼加成 |
| 朋友 | 41-60 | 战斗辅助 |
| 知己 | 61-80 | 高效双修 |
| 伴侣 | 81-95 | 强力buff |
| 道侣 | 96-100 | 最高加成 |

### 8.5 仙侣功能
- **双修**: 与仙侣一起修炼，获得额外修为加成
- **并肩作战**: 联手对敌，伤害叠加
- **赠送礼物**: 提升好感度
- **探访**: 增加互动和好感
- **命运纠缠**: 影响双修效果

### 8.6 双修机制
- 双修获得修为 = 基础修为 × 好感加成 × 资质加成 × 命运加成
- 命运纠缠度越高，双修效率越高
- 仙侣拥有专属双修技能

### 8.7 仙侣管理
- 玩家最多可拥有3位仙侣
- 仙侣可遇不可求，需要缘分
- 高境界修士更易遇到高品质仙侣

## 9. 网络同步系统 (Phase 5)

### 9.1 架构设计
采用 WebSocket 实现实时双向通信，支持多玩家在线同步。

### 9.2 核心功能
| 功能 | 说明 |
|------|------|
| 玩家登录 | 支持多玩家同时在线 |
| 房间系统 | 创建/加入/离开游戏房间 |
| 实时聊天 | 公共聊天和私聊 |
| 状态同步 | 玩家属性实时同步 |
| 动作广播 | 战斗、修炼等动作实时广播 |

### 9.3 消息协议
- **LOGIN/LOGOUT**: 登录登出
- **CREATE_JOIN/LEAVE_ROOM**: 房间管理
- **CHAT/WHISPER**: 聊天消息
- **ACTION**: 玩家动作同步
- **SYNC**: 游戏状态同步
- **HEARTBEAT**: 心跳保活

### 9.4 数据结构
- **PlayerState**: 玩家状态（境界、灵气、位置等）
- **Room**: 房间信息（玩家列表、状态）
- **GameMessage**: 通用消息格式

### 9.5 同步策略
- **全量同步**: 定期发送完整状态
- **增量同步**: 动作触发变更时发送
- **本地预测**: 客户端先执行动作，再等待服务器确认

### 9.6 技术栈
- Python asyncio
- WebSocket (ws)
- 房间管理/状态机
