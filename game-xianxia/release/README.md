# 仙途 (Xian Tu) - 修仙游戏发布包

## 游戏简介

仙途是一款文字冒险/放置修仙游戏，玩家可以体验完整的修仙历程，从炼气期到化神期，修炼功法、收服弟子、驯养灵兽、寻找仙侣，最终飞升成仙。

## 文件说明

- `xiantu_mac` - Mac 单机版可执行文件
- `xiantu_server_mac` - Mac 服务器版可执行文件
- `game_data.db` - 游戏数据库（首次运行自动创建）

## 运行方式

### 单机版
```bash
./xiantu_mac
```

### 服务器版
```bash
# 启动服务器（默认端口 8765）
./xiantu_server_mac

# 指定端口
./xiantu_server_mac --port 8080

# 查看帮助
./xiantu_server_mac --help
```

## 系统要求

- macOS 10.13+
- 无需额外安装 Python 或依赖库

- ✅ 境界修炼系统（炼气、筑基、金丹、元婴、化神）
